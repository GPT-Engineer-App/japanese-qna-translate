# japanese-qna-translate

Translate the in-app language of the Q&A bulletin board app to Japanese, ensuring all user interface elements, prompts, and instructions are fully translated to provide a native user experience for Japanese-speaking users.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/japanese-qna-translate.git
cd japanese-qna-translate
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
